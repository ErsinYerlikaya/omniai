At Omni AI Solution, innovation is at the core of what we do. This section showcases patented technologies — each designed to solve real-world challenges through intelligent, data-driven solutions.

Patent 1: Quality Assessment of Traffic Service Providers

Inventor: Dr. Yildirim Dülgar et al.
Link: DE102020005135A1

This patent presents a method for measuring the quality of a traffic service provider by analyzing how many vehicles actively send their location and time data during a traffic jam.

➤ Core Idea

  • Vehicles send GPS and timestamp data to a central server
  • The system identifies if two vehicles are stuck in the same traffic jam
  • Calculates various time, distance, and coverage ratios

➤ Why Is This Important?

  • Provides real-time data coverage score for providers
  • Enables adaptive, smart systems

➤ Key Benefits

  • Transparent traffic data evaluation
  • Boosts automation and trust in traffic platforms

Patent 2: Comparative Method for Vehicle Coverage Based on Position Data

Inventor: Dr. Yildirim Dülgar et al.
Link: DE102023002474A1

This patent introduces a method to compare vehicle coverage using real-time GPS data across different road segments using circle-based zones and normalization.

➤ Core Idea

  • Spatio-temporal data aggregation
  • Relative comparison using circle coverage method

➤ Why Is This Important?

  • Supports regional optimization and fair comparisons
  • Helps smart mobility planning

➤ Key Benefits

  • Compares coverage across segments
  • Boosts infrastructure design and AV logic

Patent 3: Cooperative Merge Support for Autonomous Vehicles

Inventor: Dr. Yildirim Dülgar
Link: DE102020002993A1

This patent supports autonomous merging behavior on multi-lane roads using V2X communication and vehicle coordination via a central system.

➤ Core Idea

  • AV notifies server of merge intent
  • Nearby cars create and protect merge slot

➤ Why Is This Important?

  • Addresses mixed traffic challenges
  • Improves traffic safety during merging

➤ Key Benefits

  • Enables reliable AV integration
  • Boosts coordination & communication

Patent 4: Smart Horn Signal Detection for Safer Roads

Inventor: Dr. Yildirim Dülgar et al.
Link: DE102020004014A1

This system captures, timestamps, and analyzes horn signals to identify high-stress areas and improve navigation safety.

➤ Core Idea

  • Detect horn usage acoustically
  • Store & analyze data in cloud

➤ Why Is This Important?

  • Adds context for autonomous decisions
  • Improves fleet safety understanding

➤ Key Benefits

  • Lightweight solution for smart cities
  • Real-time driver feedback and rerouting
